# opensea-monitor
Monitor OpenSea event and send message to Discord, with custom filters.

## Features

- Filter when monitor
  - Projects' ranking
  - NFT price
  - NFT properties
- Filter when alarm
  - No robots offer
- Message channel
  - Discord
  - DingTalk
  
## Command

- Listen: listen to OpenSea event, save it MongoDB for cache
- Dispatch: send messages to individual clients
